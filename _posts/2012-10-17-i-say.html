状态 

    最近一直在酱油，不是因为没事情，而是不想做事情，不想做事情这件事是很可怕的，就像两个人吵架，当有一方说"我就....",恐怕这个事情就变的不那么人类了，因为人类是逻辑和思考的，所以呢，我还是找一下不想做事情的理由：
     1：我的颈椎病，这个是最不借口的，我就提前说了，每天到下午四点，就开始熬不住了，头晕，恶心，有打架的冲动，奈何头晕的没力气。
     2：工作技术含量比较低，这个我也没办法，虽然说这个绝对是借口，可我就这么说了，关你毛事儿。我总结了一下下，我这两个半月的工作，如果我认真，有一周足够完成了，但是呢，我就是耗了两个月，还完成的质量稀烂，我就这样了，你开除我吧！
     3：我老了，缺乏了激情，我以前是看不得别人比我强的，现在爱谁牛逼就牛逼去，关我毛事儿，没激情就没动力就不想工作就酱油。
OK,理由这么充分就继续写下去

读书 

   最近读了两本书：1：别闹了，费曼先生 2：给加西亚的一封信
   写一下感想：
   如果倒退五年以上，我读费曼，一定激动的睡不着脚，因为那个时候我是一个怀揣梦想的青年，我有使不完的劲，我骄傲并且自大，谁都不服，我认为自己可能是个天才，最起码也是个人才，只要努力什么都有可能，我现在枯萎了，看见很多三十几岁的人还在力荐这本书，说明我这个27岁的青年老的比较快，比较明显。
   我读加西亚，很符合我的风格，我在大学毕业的时候就是这么想，给自己两年的时光，追上那些我落下的东西，不计报酬，勇往直前，可是，我现在却在心安理得的酱油。撞个豆腐死了算了。
   奥，另外啊，我不是没有收获，只是因为我今天想贬低一下自己，一直消极的说这些事情，真的

    我抽时间搞了一下我的博客，增加了PPT的支持，增加了分类等等，想找志同道合的人一起写作，可是无功而饭，那帮家伙没劲极了。
    我有一些想法，还是不想扔了C，LUA，想继续学习一下Python，还想搞一下GO，所以呢，今天就想了这么一个东西出来，既：”读书不如写作“
    关于写作有如下想法，写一本 learn go the hard way，写的时候参考众多资料，写作的风格依附learn code the hard way，所以呢读两本书 learn python the hard way和learn C the hard way，间接的巩固Pythoon和C语言。这个想法甚好，我写这个也不想出版什么的，写好之后开源它，大概2012年完成初稿。这周把大纲定义下来。

工作 

    工作是我的衣食父母啊，我最重要的还是这个，所以放到最后，之歌言简意赅的说几点，而且一定做到的：
    1：每周至少三天的努力工作，每天09：00-12：00 and 14：00-17：30是要专心工作的，不分心做别的事情
    2：质量一定要好，主动的承担灰色地带的事情
    3：对项目整体的把握，对其他项目的了解
    4：心态要好，宠辱不惊
    5：多交流，写备注和周报，养成这些操蛋的习惯
